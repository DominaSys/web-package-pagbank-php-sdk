# Solicitar homologação

A homologação valida se a sua integração com o PagBank está funcionando corretamente antes de ir para produção.

## A homologação é obrigatória

Em geral, todos os clientes passam por esse processo.

## Segurança adicional

Para aumentar a segurança da integração, é recomendado usar controles adicionais, como `reCaptcha`, na tela de checkout.

## Como solicitar

Para solicitar homologação, abra o formulário oficial de solicitação do PagBank.

## O que acontece depois

Após a análise e aprovação, a integração fica habilitada para o ambiente de produção.
