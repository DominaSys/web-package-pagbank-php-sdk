# Cartões de teste

Os cartões de teste permitem simular diversos cenários no ambiente Sandbox.

## Chaves de API

Ao usar cartões de teste, utilize chaves de API de teste nas requisições.

## Números fictícios

Os números de cartão apresentados nessa documentação são fictícios e não devem ser usados em produção.

## Exemplos de request e response

Os exemplos de resposta ajudam a validar cenários de:

- autorização bem-sucedida;
- autorização negada.

## Recursos cobertos

Os cartões de teste podem ser usados em fluxos de cartão de crédito e em simulações que dependem de resposta controlada.

Use esta página em conjunto com o simulador para cobrir cenários mais amplos de testes.
