# Testar integração

Antes de colocar uma integração em produção, use o ambiente Sandbox para validar comportamento, resposta e cenários de pagamento.

## Objetivo

A etapa de testes ajuda a confirmar se a integração foi feita corretamente e se os pagamentos estão sendo processados sem erros.

## Formas de teste

O PagBank oferece dois caminhos principais:

- cartões de teste;
- simulador.

## Finalização

Depois de validar os testes, siga para o processo de homologação.
